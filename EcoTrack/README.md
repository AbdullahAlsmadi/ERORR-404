# 🌱 EcoTrack: Smart Carbon Footprint Analyzer

> **Track 3: Carbon Footprint & Sustainable Living**  
> Developed during the 48-Hour Sustainability Hackathon at Düzce University.

## 📌 Project Overview
EcoTrack is a dynamic and gamified web application designed to help individuals and institutions track, analyze, and minimize their carbon footprint. Moving beyond traditional static calculators, our platform integrates interactive dashboards and data simulation to make sustainability engaging and actionable.

## ✨ Key Features
*   **Interactive Carbon Calculator:** Precisely calculates daily emissions based on energy usage, transportation, and lifestyle choices.
*   **Data-Driven Dashboard:** Visualizes carbon emission trends over time using interactive charts.
*   **Gamification & Leaderboard:** Encourages eco-friendly habits by ranking users or university departments based on their sustainability scores.
*   **Actionable Insights:** Provides customized recommendations to reduce unnecessary energy consumption.

## 🛠️ Tech Stack
*   **Frontend:** HTML5, CSS3, JavaScript (UI/UX focused)
*   **Backend / Logic:** Python (Data processing and emission algorithms)
*   **Version Control:** Git & GitHub
